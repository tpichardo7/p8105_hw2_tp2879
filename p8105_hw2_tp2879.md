## Proportion of Entrances Without Vending that Allow Entry

``` r
total_entrances = nyc_transit_df |>
  filter(vending == "NO") |>
  nrow()

entrances_with_entry = nyc_transit_df |>
  filter(vending == "NO", entry == TRUE) |>
  nrow()

proportion = entrances_with_entry / total_entrances
```
